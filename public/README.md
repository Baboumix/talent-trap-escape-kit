# Images du livre à ajouter

L'app attend deux images de couverture du livre ici :

- `/public/book-fr.png` — couverture française ("Tout le monde pense que tu as réussi. Sauf toi.")
- `/public/book-en.png` — couverture anglaise ("Everyone thinks you've made it. Except you.")

Sauvegarde les deux images dans ce dossier avec exactement ces noms.

Les images seront affichées en largeur 140px sur les pages de résultat (Module 1 et Module 3), avec un drop-shadow pour un effet 3D.

Si les images sont absentes, l'app ne plantera pas — elle affichera juste le texte du livre sans image.
